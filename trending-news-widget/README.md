Displays trending news headlines based on recent post views.

## Description
Trending News Widget displays the most-read posts within a configurable time window.

Ideal for news websites, blogs, and magazines.

## Features:
* View-based trending detection
* Time-window filtering
* Widgetized display
* Category filtering
* Lightweight and fast

## Installation
1. Upload the plugin to `/wp-content/plugins/`
2. Activate through the Plugins menu
3. Go to Appearance → Widgets
4. Add “Trending News”

## How is trending calculated?
Posts with the highest views within the selected time window.
