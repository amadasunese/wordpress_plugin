Create a professional digital newspaper and PDF archive with a featured current edition.

== Description ==

TNO Digital Newspaper Library allows publishers, media houses, and organizations
to publish PDF-based newspapers and documents in a clean, mobile-friendly archive.

The plugin is designed with a classic newsroom layout and works seamlessly
with existing WordPress themes.

== Features ==

* Custom post type for digital editions
* Secure PDF upload via WordPress Media Library
* Featured "Current Edition" (latest published)
* Newspaper-style archive layout
* Search and filter by year and month
* Responsive design for mobile and desktop
* Admin settings for archive display
* Theme-independent templates
* WordPress.org coding standards compliant

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to **Digital Editions â†’ Add New**
4. Upload a PDF and publish
5. Visit `/e-paper/` to view the archive

== Frequently Asked Questions ==

= Does this plugin support paid or restricted access? =
No. Version 1.0.0 is fully open. Access control may be added in future versions.

= Will this affect my existing posts or theme? =
No. The plugin is self-contained and theme-independent.

= Is it mobile-friendly? =
Yes. The layout adapts to all screen sizes.
