A professional digital newspaper and PDF archive system with a classic newsroom layout.

== Description ==

EA Digital Newspaper Library allows publishers to create a modern, mobile-friendly digital newspaper archive using PDFs.

It is designed for:
* Newspapers and media houses
* Magazines and reports
* Government and institutional publications

The plugin works out of the box with existing PDF-based newspaper posts.

== Features ==

* Custom post type for digital editions
* Secure PDF upload using WordPress Media Library
* Embedded PDF viewer with fallback
* "Today’s Edition" pinned at the top
* Classic newspaper-style archive
* Search and filtering
* Year and month collapsible archive navigation
* Mobile-responsive layout
* Settings page to control display options
* Fully theme-independent
* WordPress.org compliant

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via the Plugins menu
3. Go to **Digital Editions → Add New** to publish newspapers
4. Visit `/e-paper/` to view the archive

== Frequently Asked Questions ==

= Will this break my existing newspaper posts? =
No. The plugin uses the existing `digital_edition` post type and `_tno_pdf` meta key.

= Can I disable Today’s Edition? =
Yes. Go to **Digital Editions → Settings** and uncheck it.

= Does this work on mobile phones? =
Yes. The layout and PDF viewer are mobile-responsive.

= Can I use this for non-newspaper PDFs? =
Yes. It works for any PDF-based publication.
