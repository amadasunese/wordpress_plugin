ObserverMan Pro is a premium AI-powered assistant for WordPress newsrooms and content websites.

== Description ==

ObserverMan Pro brings contextual AI chat to your WordPress site, designed specifically for newsrooms, publishers, and premium content platforms.

It understands page context, respects paywalls, protects premium content, and helps convert engagement into subscriptions.

== Features ==

* Context-aware AI chat assistant
* Session-based conversation memory
* Rate limiting and abuse protection
* Subscription and CTA monetization engine
* Remote license validation
* Domain and IP-bound licensing
* Graceful degradation on license expiry
* Secure admin settings
