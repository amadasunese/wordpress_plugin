Create a professional digital PDF library for newspapers, reports, magazines, and documents.

== Description ==
Digital PDF Library allows you to publish and manage PDF-based content such as:

* Digital newspapers (e-papers)
* Magazines and reports
* Manuals and brochures

Features:
* Custom post type for PDFs
* Secure Media Library upload
* Responsive embedded PDF viewer
* Mobile-friendly archive grid
* View and download actions
* Fully theme-independent
